PROJECT BUSINESS SERVICE

BONZAI™ is a fictional business service created for the purpose of a master's project for course ISM 6225 (Distributed Information Systems). BONZAI™ was inspired by the Japanese meaning of Bonsai. Bonsai is an asian art form using cultivation techniques to produce small trees. Bonsai trees are known for its resilience and its ability grow in obscure places given the right conditions. Likewise, this business service BONZAI™, could provide insights to stocks.

TECHNOLOGY
Technology involved in the project development

The deployed application utilizes four endpoints from the IEX Trading APIs. The .NET Core backend was written to retrieve the desired stock information and store it into an SQL Database on Azure Cloud. (We only stored a few stock symbols to the database such as: AAPL, MSFT, KO). The .NET Core provides the API service to the front end which was wrtten in Angular 7 and utilizing HighCharts for charting. The final app is deployed to Azure Cloud.

  • Frontend : Angular 7
  • Charting : Highcharts
  • Backend : .NET Core using IEX Trading API with four end-points
  • Database : SQL Database
  • API : IEX Trading API
  • Cloud : Azure
